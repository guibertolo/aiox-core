## Variation 1: Minimalist Text Logo

**Purpose:** Primary wordmark for header/branding contexts
**Use cases:** Website header, app header, business cards, emails
**Size range:** 240px+ width

### Main Prompt

```
Create a minimalist wordmark logo for "Siga Bem", a Brazilian fleet management SaaS platform.

Design requirements:
- Text: "Siga Bem" in modern, geometric sans-serif typography (suggest: Montserrat, Inter, or Poppins weight 600-700)
- Color palette: Primary #1B3A4B with optional accent in #2D6A8F
- Style: Flat design, clean, professional, corporate
- Add a subtle geometric icon to the LEFT of the text representing a road/route
- Icon inspiration: Simple curved road line, navigation arrow, or direction path (minimal, thin stroke)
- Layout: Text and icon should have clear visual hierarchy and breathing room
- Background: Transparent
- Proportions: Wide format (aspect ratio ~3:1 or 4:1)
- No shadows, no 3D effects, no gradients
- Suitable for SVG export with clean vector paths

Style reference: Modern SaaS logos like Notion, Figma, or Slack—simple, bold, recognizable
```

### Negative Prompt

```
Avoid:
- Overly decorative or script fonts
- Realistic or photo-based elements
- Drop shadows or 3D effects
- Gradients or color blends
- Busy or cluttered composition
- Cartoon or illustrative truck imagery
- Multiple colors beyond primary palette
- Small details that don't scale to 32px
- Brazilian flags or stereotypical elements
```

### Application Notes

- **Primary header use:** Website hero, landing page, navigation bar
- **Variants:** Dark on white, white on dark, single-color
- **Minimum safe size:** 240px width
- **Scalability:** Must remain legible at 72px and clear at 1200px
- **File formats:** SVG preferred, PNG at 1x and 2x DPI
- **Usage:** Logo + wordmark combination for full branding

```
Create a ultra-simplified geometric icon suitable as a favicon and app launcher icon for "Siga Bem" fleet management app.

```

- Gradients, shadows, or effects
- Text or letters

## Export Guidelines

### SVG Specifications
- **ViewBox:** Always define viewBox (not fixed width/height)
- **Stroke width:** Use proportional units or percentage-based scaling
- **Path optimization:** Run through SVGO before delivery
- **Color format:** Use hex codes (#1B3A4B) not RGB or named colors
- **Font embedding:** Avoid web fonts—convert text to paths or use system fonts

### PNG Specifications
- **Format:** 24-bit PNG with alpha transparency
- **Dimensions:** Provide 1x and 2x versions (e.g., 192x192 and 384x384)
- **Compression:** Optimize with TinyPNG or similar (under 50KB per file)
- **Color space:** sRGB (not CMYK or Adobe RGB)
- **DPI:** 72 DPI (web standard), 300 DPI (print if needed)

### Usage Licenses
- **Ownership:** Brand assets owned by Siga Bem company
- **Restrictions:** Do not alter, remix, or use without explicit permission
- **Attribution:** Not required (company logo)
- **Derivatives:** Create variations only through official design system guidelines
