## 1. Brand Strategy

### 1.1 Posicionamento

**Declaracao de Posicionamento:**

> Siga Bem e a plataforma de gestao que transforma a operacao de cegonheiros brasileiros -- de planilhas e ligacoes telefonicas para controle digital completo -- sem exigir que ninguem vire especialista em tecnologia.

**Categoria:** SaaS de gestao para transporte rodoviario de veiculos (cegonha)

**Diferencial competitivo:** Unica plataforma desenhada especificamente para o nicho de cegonheiros, com interface que respeita o perfil do usuario (motoristas 35-50 anos, baixa familiaridade digital) e linguagem do setor.

**Mapa de posicionamento:**

```
                    COMPLEXO
                       |
    ERPs genericos  ---|---  Siga Bem NAO esta aqui
                       |
  GENERICO ------------|------------- ESPECIALIZADO
                       |
    Planilhas/         |    [SIGA BEM]
    WhatsApp        ---|---  Simples + Especializado
                       |
                    SIMPLES
```

O Siga Bem ocupa o quadrante **simples + especializado** -- facil de usar como WhatsApp, mas com toda a inteligencia que o setor precisa.

### 1.2 Tom de Voz

O Siga Bem fala como um **parceiro de estrada experiente**: sabe do que esta falando, nao complica, e torce pelo seu sucesso.

| Dimensao | Posicao | Exemplo |
|----------|---------|---------|
| Formal <---> Informal | 60% informal | "Sua frota esta rendendo bem" (nao "Indicadores de performance otimizados") |
| Tecnico <---> Acessivel | 80% acessivel | "Controle suas viagens" (nao "Gestao de rotas logisticas") |
| Distante <---> Proximo | 70% proximo | "Voce" (nunca "o usuario" ou "o senhor") |
| Serio <---> Descontraido | 60% serio | Profissional com calor humano, sem piadas forcadas |

**Vocabulario da marca:**

| Use | Evite |
|-----|-------|
| Viagem | Rota logistica |
| Frota | Ativos de transporte |
| Motorista | Operador |
| Cegonha | Veiculo transportador |
| Controle | Dashboard / Painel |
| Ganho / Resultado | ROI / Metricas |

**Expressoes permitidas da cultura rodoviaria:**
- "Siga bem" (despedida/desejo)
- "Estrada" como metafora para jornada
- "Parceiro de estrada" para se referir ao usuario
- "Roda girando" para progresso/operacao ativa

### 1.3 Valores da Marca (5 Pilares)

```
    CONFIANCA          SIMPLICIDADE         RESPEITO
    "Sua carga         "Tecnologia que      "Feito pra
     esta segura"       faz sentido"         quem e da
                                             estrada"
         \                  |                  /
          \                 |                 /
           +--------[SIGA BEM]--------+
          /                 |                 \
         /                  |                  \
    RESULTADO           PARCERIA
    "Sua frota          "Cresce junto
     rende mais"         com voce"
```

| Pilar | Significado | Como se manifesta |
|-------|-------------|-------------------|
| **Confianca** | Dados seguros, sistema estavel, zero surpresa | Uptime visivel, backups automaticos, transparencia total |
| **Simplicidade** | Qualquer motorista consegue usar | Telas limpas, fluxos curtos, linguagem clara |
| **Respeito** | Entendemos a realidade do cegonheiro | Funciona no celular, offline-first, sem exigir email corporativo |
| **Resultado** | A frota rende mais com o Siga Bem | Relatorios de ganho, comparativos, alertas de custo |
| **Parceria** | Nao e um fornecedor, e um parceiro | Suporte humano, onboarding assistido, preco justo |

### 1.4 Taglines / Slogan

**Opcao principal (recomendada):**

> **"Siga Bem. Sua frota no controle."**

Justificativa: Une o nome da marca (que ja e uma expressao conhecida) com a promessa central (controle). Curto, memoravel, funciona em todas as aplicacoes.

**Opcao alternativa 1:**

> **"Siga Bem. Gestao de cegonha do jeito certo."**

Justificativa: Explicita o nicho (cegonha) e o diferencial (do jeito certo = simples, especializado). Melhor para materiais onde o publico ainda nao conhece a marca.

**Opcao alternativa 2:**

> **"Estrada livre, frota no controle."**

Justificativa: Mais emocional, evoca a liberdade da estrada com a seguranca do controle. Funciona bem em campanhas e splash screens.

### 1.5 Personalidade da Marca

Se o Siga Bem fosse uma pessoa, seria um **caminhoneiro experiente que virou gestor** -- conhece a estrada, fala a lingua, mas aprendeu a organizar tudo com metodo.

| Traco | Intensidade | Manifestacao |
|-------|-------------|--------------|
| Confiavel | Alta | Nunca promete o que nao entrega |
| Pratico | Alta | Vai direto ao ponto |
| Acolhedor | Media | Recebe bem, tem paciencia |
| Moderno | Media | Usa tecnologia, mas nao ostenta |
| Ambicioso | Baixa | Quer crescer, mas sem atropelar |

**Arquetipos de marca:**
- Primario: **O Companheiro** (pertencimento, confianca, parceria)
- Secundario: **O Heroi** (superacao, resultado, eficiencia)

## 2. Logo Concepts

### 2.1 Conceito A -- Minimalista/Tech

**Nome interno:** "Rota Digital"

**Filosofia:** Posicionar o Siga Bem como plataforma tecnologica moderna, usando formas geometricas puras que comunicam precisao e inovacao. Ideal para investidores e mercado tech.

**Formato:** Combinacao (icone + logotipo)

**Elementos visuais:**
- Icone: Letra "S" estilizada formada por duas setas conectadas (uma apontando para frente/direita representando "siga", outra curvando para cima representando "bem"/progresso). As setas se conectam formando um caminho continuo, sugerindo rota e fluxo
- As setas usam cantos arredondados (border-radius) para suavidade
- Espessura uniforme das linhas (stroke-based, nao preenchido)
- O espaco negativo entre as setas forma sutilmente a silhueta de uma estrada em perspectiva

**Paleta aplicada:**
- Icone: `#1B3A4B` (primary) com gradiente sutil para `#2D6A8F` (primary-700)
- Logotipo: `#1B3A4B` solido
- Fundo: branco ou transparente

**Tipografia:**
- Logotipo: **Inter Bold** para "Siga", **Inter Medium** para "Bem"
- Tracking: +2% para respiracao
- "Siga" e "Bem" na mesma linha, separados por espaco

**Variacoes:**
- **Horizontal:** Icone a esquerda + "Siga Bem" a direita (uso principal)
- **Vertical (empilhada):** Icone acima + "Siga Bem" abaixo (redes sociais)
- **Icone solo:** Apenas o icone "S" com setas (app icon, favicon)
- **Monocromatica:** Versao em `#1B3A4B` puro, sem gradiente
- **Negativa:** Branco sobre fundo `#1B3A4B`

**Tamanho pequeno (favicon/app icon):**
- Apenas o icone "S" com setas, sem texto
- Funciona bem ate 16x16px pois as formas sao simples e geometricas
- Em tamanhos muito pequenos, o gradiente e removido (versao solida)

**Pontos fortes:** Escalabilidade, modernidade, versatilidade digital
**Pontos fracos:** Pode parecer generico demais para o setor rodoviario; donos de frota mais velhos podem nao se identificar imediatamente

### 2.2 Conceito B -- Rodoviario/Transporte

**Nome interno:** "Estrada Segura"

**Filosofia:** Comunicar pertencimento ao universo rodoviario brasileiro. O motorista olha e reconhece: "isso e pra mim". Usa elementos concretos do setor para gerar identificacao imediata.

**Formato:** Combinacao (icone emblematico + logotipo)

**Elementos visuais:**
- Icone: Silhueta simplificada de um caminhao cegonha visto de perfil (cabine + carroceria com 2 niveis para veiculos), integrada a uma forma de escudo/badge arredondado
- O escudo tem a parte inferior aberta, formando uma estrada que "sai" do emblema
- Uma linha fina de estrada (tracejada, estilo rodovia) passa por baixo do caminhao
- O conjunto transmite protecao (escudo) + movimento (estrada) + setor (cegonha)

**Paleta aplicada:**
- Escudo/caminhao: `#1B3A4B` (primary)
- Estrada/detalhes: `#2D6A8F` (primary-700)
- Acento opcional: `#B8860B` (warning/dourado) para um detalhe no escudo (faixa ou borda) comunicando solidez/tradicao
- Logotipo: `#1B3A4B`

**Tipografia:**
- Logotipo: **Inter Bold** para ambas as palavras
- Estilo mais compacto, tracking normal
- Opcao de tagline abaixo em **Inter Regular** tamanho menor

**Variacoes:**
- **Horizontal:** Escudo/icone a esquerda + "Siga Bem" a direita + tagline opcional abaixo
- **Vertical (empilhada):** Escudo acima + "Siga Bem" abaixo
- **Icone solo:** Escudo com silhueta do caminhao (sem texto)
- **Monocromatica:** Tudo em `#1B3A4B`, sem dourado
- **Negativa:** Branco sobre fundo escuro, escudo com borda branca

**Tamanho pequeno (favicon/app icon):**
- Versao simplificada do escudo: apenas a forma do escudo com a silhueta do caminhao ultra-simplificada (retangulo + cabine)
- Remove a estrada tracejada em tamanhos < 32px
- Legivel ate 16x16px com a simplificacao

**Pontos fortes:** Identificacao imediata com o setor, gera confianca, comunica tradicao e solidez
**Pontos fracos:** Pode parecer "pesado" demais para uma plataforma digital; escalabilidade limitada em tamanhos muito pequenos; risco de parecer datado

### 2.3 Conceito C -- Hibrido (RECOMENDADO)

**Nome interno:** "Caminho Digital"

**Filosofia:** O melhor dos dois mundos. Formas modernas e limpas que incorporam sutilmente a essencia do transporte rodoviario. O dono da frota se identifica, o motorista entende, e o produto parece profissional e atual.

**Formato:** Combinacao (icone + logotipo)

**Elementos visuais:**
- Icone: Uma forma compacta que combina:
  - Contorno externo em formato de "pin" de mapa/localizacao (representando rota, destino, tecnologia de rastreamento)
  - Dentro do pin, a silhueta minimalista de um caminhao cegonha formada por linhas geometricas simples (3-4 linhas: cabine + 2 niveis da carroceria)
  - O caminhao dentro do pin "aponta" para a direita (direcao = progresso, "siga")
- A base do pin se transforma em uma seta apontando para baixo/frente, sugerindo "aqui esta voce, siga em frente"
- Linhas com peso uniforme, cantos levemente arredondados (raio 2-4px em proporcao)
- Espaco negativo limpo, icone funciona como forma unica

**Paleta aplicada:**
- Pin/contorno: `#1B3A4B` (primary)
- Caminhao interno: `#2D6A8F` (primary-700)
- Acento sutil: `#2D6A4F` (success/verde) na base do pin ou como ponto, comunicando "tudo certo, siga bem"
- Logotipo: `#1B3A4B`

**Tipografia:**
- "Siga" em **Inter Bold** -- peso, acao, comando
- "Bem" em **Inter Medium** -- leveza, desejo, cuidado
- O contraste de peso entre as duas palavras cria ritmo visual e reforcar a dupla leitura do nome (imperativo + desejo)
- Tracking: +1.5%

**Variacoes:**
- **Horizontal (principal):** Pin-icone a esquerda + "Siga Bem" a direita. Proporcao icone:texto = 1:2.5
- **Horizontal com tagline:** Mesma horizontal + tagline abaixo do logotipo em Inter Regular 50% do tamanho
- **Vertical (empilhada):** Pin-icone acima + "Siga Bem" abaixo (perfil de redes sociais, splash screen)
- **Icone solo:** Apenas o pin com caminhao (app icon, favicon, marca d'agua)
- **Monocromatica:** Tudo em `#1B3A4B`, sem acento verde
- **Negativa:** Branco sobre fundo `#1B3A4B` ou `#0D1B2A` (escuro profundo)
- **Versao simplificada ultra-pequena:** Pin com forma interna reduzida a retangulo simples (< 24px)

**Tamanho pequeno (favicon/app icon):**
- Icone solo (pin + caminhao simplificado) ate 32x32px com detalhes completos
- Abaixo de 24px: pin com retangulo interno (remove detalhes do caminhao)
- 16x16px (favicon): apenas silhueta do pin com preenchimento solido
- Todas as versoes mantem o ponto verde de acento como indicador de "ativo/online"

**Pontos fortes:** Moderno sem perder identidade setorial; escalavel; memoravel; o pin de localizacao conecta com rastreamento/gestao de frota; funciona em todos os tamanhos
**Pontos fracos:** Requer mais cuidado na execucao para equilibrar os elementos

**Por que este e o recomendado:**
1. O pin de localizacao e universalmente entendido (Google Maps, Waze, Uber) -- os motoristas ja conhecem esse simbolo
2. A cegonha dentro do pin gera identificacao setorial sem parecer datado
3. O acento verde (`#2D6A4F` success) comunica "tudo certo" sem palavras
4. Funciona de 16px (favicon) ate outdoor
5. A dualidade tech + transporte reflete exatamente o posicionamento da marca

## 3. Brand Guidelines

### 3.1 Uso Correto da Logo

**CORRETO:**
- Logo na cor original sobre fundo branco ou claro (< 20% de cinza)
- Logo negativa (branca) sobre fundo `#1B3A4B` ou mais escuro
- Logo monocromatica em preto sobre fundo claro
- Logo monocromatica em branco sobre fundo escuro
- Logo com respiro adequado (ver 3.2)
- Tagline sempre abaixo, nunca ao lado, nunca acima

**INCORRETO:**
- Nunca rotacionar a logo
- Nunca distorcer proporcoes (esticar horizontal ou verticalmente)
- Nunca aplicar sombra, brilho, ou efeitos 3D
- Nunca usar sobre fundos com padrao complexo ou foto sem overlay escuro (min 60% opacidade)
- Nunca mudar as cores da logo (exceto versoes mono/negativa oficiais)
- Nunca separar o icone do logotipo com distancia diferente da especificada
- Nunca recriar o logotipo com outra fonte
- Nunca colocar a logo dentro de formas geometricas nao oficiais (circulos, losangos, etc.)
- Nunca usar a versao horizontal onde a vertical cabe melhor (e vice-versa)

### 3.2 Area de Respiro (Safe Zone)

A area de respiro minima ao redor da logo e igual a **1x a altura do icone (pin)** em todas as direcoes.

```
         +-----------------------------------------+
         |                                         |
         |    [1x]                                 |
         |     |                                   |
         |  [1x]--[ICONE]--[Siga Bem]--[1x]       |
         |     |                                   |
         |    [1x]                                 |
         |                                         |
         +-----------------------------------------+

         Onde 1x = altura do icone pin
```

Para a versao vertical (empilhada):

```
         +---------------------+
         |                     |
         |    [1x]             |
         |     |               |
         |  [1x]--[ICONE]--[1x]  |
         |     |               |
         |  [Siga Bem]         |
         |     |               |
         |    [1x]             |
         |                     |
         +---------------------+
```

Nenhum elemento grafico, texto ou borda deve invadir a safe zone.

### 3.3 Tamanhos Minimos

| Variacao | Largura minima | Contexto |
|----------|---------------|----------|
| Horizontal (icone + texto) | 120px digital / 30mm impresso | Headers, documentos |
| Vertical (empilhada) | 80px digital / 20mm impresso | Redes sociais, cards |
| Icone solo (detalhado) | 32px digital / 8mm impresso | App icon, avatares |
| Icone solo (simplificado) | 16px digital / 4mm impresso | Favicon |

Abaixo do tamanho minimo, usar sempre a versao simplificada ou o icone solo.

### 3.4 Aplicacoes

#### App Icon PWA (192x192 e 512x512)

- Usar **icone solo** (pin com caminhao)
- Fundo: `#1B3A4B` (primary)
- Icone: branco (`#FFFFFF`)
- Acento: `#2D6A4F` (ponto verde)
- Border-radius: seguir padrao do sistema (Android arredonda automaticamente, iOS usa squircle)
- Padding interno: 15% em cada lado (o icone ocupa 70% da area)

```
+---------------------------+
|        #1B3A4B            |
|                           |
|     +---------------+     |
|     |               |     |
|     |   [PIN COM    |     |
|     |   CAMINHAO    |     |
|     |   BRANCO]     |     |
|     |       *verde  |     |
|     +---------------+     |
|                           |
+---------------------------+
```

#### Favicon (16x16, 32x32, 48x48)

- 48x48: Pin com caminhao simplificado, fundo `#1B3A4B`
- 32x32: Pin com retangulo interno, fundo `#1B3A4B`
- 16x16: Pin preenchido solido branco sobre `#1B3A4B`
- Formato: `.ico` com multiplas resolucoes + `.svg` para navegadores modernos

#### Header do Site / App

- Versao horizontal (icone + texto)
- Altura do logo no header: 32-40px
- Cor: `#1B3A4B` sobre fundo claro, branco sobre header escuro
- Alinhamento: esquerda, verticalmente centralizado no header
- Safe zone respeitada em relacao a itens de navegacao

#### Sidebar do App

- Sidebar expandida: versao horizontal, largura 160px
- Sidebar colapsada: icone solo, 32x32px
- Transicao suave entre as duas versoes (fade/slide)

#### Email / Assinatura

- Versao horizontal, largura maxima 180px
- Sempre em PNG com fundo transparente
- Resolucao: 2x para telas retina (360px real, exibido a 180px)

#### Nota Fiscal / Documentos

- Versao horizontal monocromatica (preto)
- Posicao: canto superior esquerdo
- Tamanho: 30-40mm de largura
- Acompanhada de CNPJ e dados legais abaixo

#### Login Screen

- Versao vertical (empilhada) centralizada
- Tamanho do icone: 64-80px
- "Siga Bem" abaixo em 24-32px
- Tagline abaixo em 14px, cor `#6B7280` (cinza medio)
- Fundo: branco ou gradiente sutil de `#F8FAFC` para `#E2E8F0`

### 3.5 Versoes Oficiais

| Versao | Uso | Fundo |
|--------|-----|-------|
| **Colorida principal** | Uso padrao em materiais digitais e impressos | Branco / claro |
| **Monocromatica preta** | Documentos, notas fiscais, carimbos | Branco |
| **Monocromatica branca** | Nao usar isolada (baixo contraste) | N/A |
| **Negativa (branca sobre escuro)** | Headers escuros, fundos fotograficos com overlay | `#1B3A4B` ou escuro |
| **Negativa com acento** | Splash screen, hero sections | `#1B3A4B` com ponto `#2D6A4F` |

## Anexo B -- Contraste WCAG

| Combinacao | Ratio | WCAG AA | WCAG AAA |
|-----------|-------|---------|----------|
| `#1B3A4B` sobre `#FFFFFF` | 10.2:1 | PASSA | PASSA |
| `#FFFFFF` sobre `#1B3A4B` | 10.2:1 | PASSA | PASSA |
| `#2D6A8F` sobre `#FFFFFF` | 4.8:1 | PASSA | FALHA |
| `#2D6A4F` sobre `#FFFFFF` | 5.4:1 | PASSA | FALHA (texto grande: PASSA) |
| `#B8860B` sobre `#FFFFFF` | 3.9:1 | FALHA (usar apenas em elementos grandes/decorativos) | FALHA |
| `#C41E3A` sobre `#FFFFFF` | 5.1:1 | PASSA | FALHA |
| `#1B3A4B` sobre `#F8FAFC` | 9.8:1 | PASSA | PASSA |
| `#64748B` sobre `#FFFFFF` | 4.6:1 | PASSA | FALHA |

**Nota importante:** `#B8860B` (warning/dourado) NAO passa WCAG AA para texto normal. Usar apenas como cor de fundo com texto escuro, ou como elemento decorativo/iconografico, nunca como cor de texto sobre fundo branco.
