## Como Usar

Cada arquivo e self-contained (HTML + CSS + JS inline). Para usar:

1. Abrir o `.html` no navegador para preview
2. Copiar o CSS/HTML relevante para seu projeto
3. Incluir o StringTune via CDN: `<script src="https://unpkg.com/@fiddle-digital/string-tune@1.1.53/dist/index.js"></script>`
