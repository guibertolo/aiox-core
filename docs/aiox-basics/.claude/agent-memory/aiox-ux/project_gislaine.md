Projeto site da Dra. Gislaine Rodrigues, advogada de Direito de Familia em Sao Bernardo do Campo/SP.

**Why:** Presenca digital profissional para captacao via Google e WhatsApp. OAB/SP 456.789.

**Stack:** Next.js 15 App Router + TypeScript + Tailwind CSS 3.4 + MDX + Vercel.

**Paleta aprovada (Aria):** navy (azul marinho profundo), gold (dourado terracota), stone (cinza quente).

**Principio UX central:** Publico em estado emocional vulneravel (divorcio, guarda, pensao). Design precisa: acolher + confiar + agir. WhatsApp acessivel em 2 cliques.

**Conformidade OAB Provimento 205/2021:** Sem precos, sem promessas de resultado, sem comparativos, sem captacao ativa.
