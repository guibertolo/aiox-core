Setup Claude Code no VS Code

Guia completo para configurar o Claude Code do zero no Windows. Passo a passo para iniciantes.

O que voce precisa ter antes de comecar

  • VS Code instalado (voce ja tem!)
  • Node.js 18+ instalado (vamos verificar no Passo 1)
  • Conta na Anthropic com plano Claude Pro, Max ou Team (para ter acesso ao Claude Code)
  • Conexao com a internet (obvio, mas vale lembrar)

Indice

  1. Instalar o Node.js
  2. Instalar o Claude Code (CLI)
  3. Fazer login no Claude Code
  4. Instalar a extensao no VS Code
  5. Configurar o idioma para portugues
  6. Configurar permissoes (modo YOLO)
  7. Testar se tudo funciona
  8. Dicas de uso no dia a dia
1
Instalar o Node.js
~5 min

O Claude Code precisa do Node.js para funcionar. Primeiro, vamos verificar se voce ja tem instalado.

a

Abra o terminal do VS Code (atalho: Ctrl + ' ou pelo menu Terminal > New Terminal).

b

Digite este comando para verificar se o Node.js esta instalado:

node --version

Se aparecer algo como v18.x.x, v20.x.x ou v22.x.x, voce ja tem! Pule para o Passo 2.

c

Se deu erro ou nao reconheceu o comando, voce precisa instalar:

  1. Acesse nodejs.org
  2. Clique no botao verde "Download Node.js (LTS)"
  3. Execute o instalador (.msi) que baixou
  4. Clique Next, Next, Next, Install (pode deixar tudo padrao)
  5. Feche e reabra o VS Code depois de instalar
  6. Teste novamente com node --version
Importante: Depois de instalar o Node.js, voce PRECISA fechar e reabrir o VS Code para ele reconhecer o comando. Se nao funcionar mesmo depois de reabrir, reinicie o computador.
2
Instalar o Claude Code (CLI)
~2 min

Agora vamos instalar o Claude Code globalmente no seu computador.

a

No terminal do VS Code, execute:

npm install -g @anthropic-ai/claude-code

Aguarde terminar (pode levar 1-2 minutos dependendo da internet).

b

Verifique se instalou corretamente:

claude --version

Deve aparecer algo como 2.x.x (Claude Code).

Se der erro de permissao: No Windows, abra o terminal como Administrador. Clique com botao direito no VS Code > "Executar como administrador", e tente o comando novamente.
3
Fazer login no Claude Code
~2 min

Voce precisa autenticar sua conta Anthropic para usar o Claude Code.

a

No terminal, digite:

claude

Isso vai abrir o Claude Code pela primeira vez.

b

Ele vai perguntar como voce quer autenticar. Escolha:

# Use as setas do teclado para selecionar e Enter para confirmar > Anthropic (claude.ai account)
c

Vai abrir uma pagina no navegador para voce fazer login na sua conta Anthropic. Faca o login normalmente e autorize o acesso.

d

Depois de autorizar, volte ao terminal. Deve aparecer uma mensagem de sucesso e o prompt do Claude Code.

Digite /exit para sair por enquanto (vamos configurar mais coisas antes de usar).

Sobre o plano: O Claude Code precisa de um plano pago da Anthropic (Pro $20/mes, Max $100/mes ou $200/mes). O plano gratuito NAO inclui acesso ao Claude Code.
4
Instalar a extensao no VS Code
~1 min

A extensao do Claude Code integra ele direto na interface do VS Code, com um painel lateral dedicado.

a

Abra o VS Code e pressione Ctrl + Shift + X para abrir o painel de extensoes.

b

Na barra de busca, digite:

Claude Code
c

Procure a extensao oficial da Anthropic (nome: "Claude Code", autor: "Anthropic"). Clique em Install.

d

Depois de instalar, voce vai ver o icone do Claude na barra lateral esquerda do VS Code. Clique nele para abrir o painel.

Onde fica no VS Code

Depois de instalar, o Claude aparece na barra lateral esquerda (onde ficam os icones de Explorer, Search, Git, etc). Clique no icone do Claude para abrir o chat integrado. Voce tambem pode usar o atalho Ctrl + Shift + P e digitar "Claude" para ver todos os comandos disponiveis.

5
Configurar o idioma para portugues
~1 min

Por padrao o Claude responde em ingles. Vamos configurar para ele responder sempre em portugues.

a

No terminal do VS Code, execute:

claude config set language portuguese
b

Para confirmar que funcionou:

claude config get language

Deve retornar portuguese.

Pronto! Agora o Claude vai responder em portugues em todas as conversas.
6
Configurar permissoes (modo autonomo)
~1 min

Por padrao, o Claude pede autorizacao para cada acao (ler arquivo, rodar comando, etc). Isso e seguro mas pode ser chato. Voce pode deixar ele mais livre.

Voce tem 2 opcoes. Escolha UMA:

Opcao A (Recomendado para iniciantes): Modo "Aceitar tudo" — o Claude pede permissao mas voce pode clicar "Allow" rapidamente.

Opcao B (Modo avancado): Modo "YOLO" — o Claude faz tudo sem perguntar. Mais rapido, mas voce precisa confiar no que ele faz.
A

Opcao A - Modo padrao (nao precisa fazer nada)

Se voce prefere ter controle, nao precisa mudar nada. O Claude vai pedir permissao e voce clica em "Allow" ou "Deny" para cada acao.

B

Opcao B - Modo YOLO (sem pedir permissao)

No terminal do VS Code, execute:

claude config set default_mode bypassPermissions
Atencao: No modo YOLO, o Claude pode executar comandos, criar/editar/deletar arquivos sem te perguntar. E muito mais produtivo, mas voce precisa prestar atencao no que ele esta fazendo. Se estiver comecando, use a Opcao A primeiro ate se sentir confortavel.
7
Testar se tudo funciona
~2 min

Vamos fazer um teste rapido para garantir que tudo esta funcionando.

a

Abra uma pasta de projeto no VS Code (ou crie uma pasta vazia para teste):

# Criar pasta de teste mkdir C:\teste-claude # Abrir no VS Code code C:\teste-claude
b

Clique no icone do Claude na barra lateral do VS Code para abrir o painel de chat.

c

Digite uma mensagem de teste:

Crie um arquivo index.html com uma pagina simples que diz "Ola Mundo"
d

O Claude deve criar o arquivo automaticamente. Verifique se o arquivo apareceu no Explorer do VS Code.

Se funcionou, parabens! Seu Claude Code esta configurado e pronto para uso.
+
Dicas de uso no dia a dia

Atalhos uteis

Ctrl + Shift + P Abrir comandos do Claude
Ctrl + L Abrir chat do Claude (atalho rapido)
Escape Parar o Claude no meio de uma resposta
/help Ver comandos disponiveis no chat

O que voce pode pedir para o Claude

  • "Crie um site..." — ele cria os arquivos HTML, CSS e JS
  • "Corrija o bug em..." — ele le o codigo e conserta
  • "Explique esse codigo" — selecione o codigo e pergunte
  • "Rode npm install e configure o projeto" — ele executa comandos no terminal
  • "Refatore para ficar mais limpo" — ele melhora o codigo existente
  • "Pesquise como fazer X" — ele pode buscar na web por voce

Comandos uteis (digite no chat)

# Comandos que voce digita direto no chat do Claude: /help # Ver todos os comandos /compact # Limpar memoria da conversa (quando ficar lento) /clear # Comecar conversa do zero /cost # Ver quanto ja gastou na sessao Escape # Parar o Claude no meio de uma acao

Boas praticas

  • 1. Seja especifico — "Crie um botao azul com bordas arredondadas" e melhor que "crie um botao"
  • 2. Abra a pasta do projeto — O Claude trabalha melhor quando voce abre a pasta do projeto no VS Code (File > Open Folder)
  • 3. Use /compact quando ficar lento — Conversas longas podem ficar pesadas, /compact resolve
  • 4. Salve antes de pedir mudancas grandes — Faca commit no Git antes de mudancas grandes, para poder voltar atras se precisar
  • 5. Voce pode arrastar arquivos — Arraste imagens ou arquivos para o chat para o Claude analisar

Tudo pronto!

Seu Claude Code esta configurado. Aqui esta o checklist final:

Node.js instalado e funcionando
Claude Code CLI instalado (claude --version)
Login feito na conta Anthropic
Extensao Claude Code instalada no VS Code
Idioma configurado para portugues
Teste "Ola Mundo" funcionou
